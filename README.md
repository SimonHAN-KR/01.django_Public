# Django를 이용한 주식정보 사이트

- 기간
    - 4일 (2021.08.17 ~ 2021.08.20)
- 개요
    - Django를 이용하여 html 화면 개발
    - 6가지 랭크화를 통하여 정보 검색
    - 네이버의 주식정보 들을 이용하여 관련 정보를 불러옴
- 맡은 업무
    - 주식코드를 이용하여 상세페이지 연결 (01.django/data/templates/data/detail.html)
    - 종목코드, 회사명, 주식그래프, 업종, 주가, 시가, 기업실적분석
- 개선점
    - 더 많은 정보 연결 필요
    - 더 깔끔한 디자인 필요
